## Software testing project 2025 - COMP.SE.200 [![Coverage Status](https://coveralls.io/repos/github/Ficelo/software-testing-project/badge.svg?branch=main)](https://coveralls.io/github/Ficelo/software-testing-project?branch=main)

This repo is the project for the software testing course at the University of Tampere.

#### How to run

Inside of the root folder :

##### To run the tests :
```bash
npm run test
```

##### To get coverage :
```bash
npm run coverage
```
